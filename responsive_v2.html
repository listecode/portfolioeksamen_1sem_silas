Grundlæggende Web

Gå til Responsive_v2

På Tema 2 : Grundlæggende Web blev vi introduceret til html og css. Vi lærte om forskellige stilarter, som neumorphisme, minimalisme og brutalisme. Vi lærte om mobile first og media queries, som er med til give sitet den rigtige opsætning i forhold til hvilken skærmstørrelse brugeren har. Mit site bygger på minimalisme og handler om at skabe overblik for brugeren og have meget få elementer.
